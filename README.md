TimeTravel Agency — Webapp Interactive
Webapp pour une agence de voyage temporel fictive,
créée avec IA générative dans le cadre d'un projet pédagogique.

🛠️ Stack technique
React + Tailwind CSS
Shadcn/ui (composants UI)
Next.js
Hébergement : Vercel

✨ Features
Landing page immersive en dark mode
Hero animé avec vortex temporel
Galerie de 3 destinations temporelles (cards interactives)
Chatbot IA "Chronos" simulé en local
Quiz de recommandation personnalisée (4 questions)
Formulaire de pré-réservation
Design responsive mobile-first
Effets glassmorphism et animations CSS

🗺️ Les 3 destinations
Paris 1889 — Belle Époque, Tour Eiffel — à partir de 12 900 €
Crétacé -65M ans — Dinosaures, nature primitive — à partir de 18 500 €
Florence 1504 — Renaissance, ateliers d'artistes — à partir de 10 800 €

🤖 IA utilisées
Génération de code : v0.dev (Vercel) — Claude Sonnet
Chatbot : simulé en local (sans API externe)
Visuels : générés lors du projet précédent

📦 Installation locale
Cloner le repo
Lancer npm install
Lancer npm run dev
Ouvrir http://localhost:3000/
